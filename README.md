# Como executar o projeto

## Front end web
Pré-requisitos: npm / yarn

```bash
# clonar repositório
git clone https://github.com/Jeann-Cavalcante/FormChakraUi.git

# instalar dependências
yarn install

# executar o projeto
yarn dev
```
